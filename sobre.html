MAJEM Software


   Nós somos um grupo de cinco aprendizes em desenvolvimento de sistemas que formaram essa "empresa" a partir do Senai. Desenvolvemos este sistema voltado para uma boutique — uma loja geralmente menor do que as demais — especializada na venda de produtos de moda e luxo, como roupas, calçados, acessórios, perfumes e até mesmo itens de decoração e design exclusivos. Cada uma de nós desenvolveu parte de uma página do sistema, desde o Front-end, que é a "cara" da página visível ao usuário, até o Back-end, que é a parte que funciona por trás do sistema. O Back-end foi programado em duas linguagens diferentes: Java e JavaScript. Já o Front-end foi desenvolvido em HTML e CSS. O objetivo da nossa empresa ao criar este sistema foi oferecer uma plataforma não apenas funcional e bonita, mas também prática de usar. O sistema, contém sete funcionalidades principais:

Cadastrar: possibilita cadastrar produtos de maneira isolada ou em lote;

Listar: mostra todos os produtos cadastrados existentes no banco de dados;

Consultar: permite visualizar todas as informações do produto por meio de seu código de identificação;

Atualizar: possibilita modificar dados como a quantidade em estoque (o código de identificação não pode ser alterado);

Apagar: permite remover um produto do sistema, caso ele não esteja mais disponível no sistema;

Gráficos: mostra de forma gráfica a quantidade e o preço dos produtos.
